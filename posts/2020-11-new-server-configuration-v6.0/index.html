云服务器配置笔记 v6.0

2020.11.18

为了之后将业务从 Windows 迁移到 Linux，现在就买了一台阿里云服务器，安装了 Linux 系统，然后开始配置各方面的功能，以便之后迅速迁移。

系统安装及配置

系统安装

选择安装 Alibaba Cloud Linux等保2.0三级版 ，这样自己不用花心思去折腾很多配置，不是最重要的地方，省心最关键。

安装时创建密钥对，这样必须通过密钥对才能远程登录系统，提高安全性。

执行加固脚本

安装完成后进入系统，按照提示执行 /root/cybersecurity.sh 进行安全加固：

  • 创建普通用户 www，并设置密码
  • 创建审计员用户 audit，并设置密码
  • 创建安全员用户 secure，并设置密码

创建完成后重新 SSH 连接服务器，发现不能用密钥对登录 www 用户了，只能用账号密码的方式登录。

执行安全更新

在终端执行 yum updateinfo list ，查看当前可用的 CVE 安全更新。

如果有可用的 CVE 安全更新，可以再执行 yum upgrade --security，来安装这些 CVE 安全更新。
